Bash and Basic Linux Commands

On the second day, we learned about the Linux shell and Bash. We also learned how files and directories are organized in Linux.

We practiced some basic Linux commands for creating, deleting, copying, and moving files and directories.

Topics covered:

- Introduction to Shell and Bash
- Linux File System Structure
- Files and Directories
- File and Directory Permissions
- Basic Linux Commands
- Redirection and Pipes

Some commands we practiced:

pwd
ls
cd
mkdir
touch
cp
mv
rm
cat
chmod

Day 3 – Programming in Linux

On the third day, we learned how to create and run programs in Linux.

We used the Nano editor to write shell scripts and learned some basic concepts of Bash scripting.

Topics covered:

- Creating files in Linux
- Using the Nano editor
- Introduction to Shell Scripts
- Variables
- Taking input using the "read" command
- Conditional Statements
- Loops
- Running Shell Scripts

Example:

#!/bin/bash

echo "Enter your name:"
read name

echo "Hello $name"

Day 4 – PC and Network Troubleshooting
